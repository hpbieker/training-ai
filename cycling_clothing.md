# Cycling Clothing

Use this as a practical wardrobe reference when choosing kit for outdoor rides.
Castelli temperature ranges are manufacturer ranges, not personal rules. Adjust
for ride intensity, wind, rain, sun, route exposure, stop risk and personal
tolerance.

## Quick Selection Rules

| Conditions | Default kit idea |
| --- | --- |
| 18-25 C, dry | Bib shorts, short-sleeve jersey, light gloves. Vest only if wind/descents/stops. |
| 15-20 C, dry, variable/coastal | Bib shorts, short-sleeve jersey, light vest in pocket or on at start. Consider knee warmers if knees get cold. |
| 10-16 C, dry | Bib shorts + knee warmers or leg warmers, warmer base layer, vest or Perfetto depending wind/intensity. |
| 5-14 C, windy/damp | Perfetto territory. Add warmer base layer and consider full leg coverage. |
| Rain risk | Emergency/Slicker rain shell carried. Wear only when rain/wind makes it worth the heat cost. |

## Today's Example: Oslo-Fjallbacka Type Ride

For a route starting mild around 17 C, warming near 19-20 C, then cooling to
15-16 C on the coast, the reasonable baseline is:

- Bib shorts.
- Short-sleeve jersey.
- Light base layer or no base layer, depending start comfort.
- Pro Light Wind Vest available for start/coastal finish.
- Knee warmers only if cool knees are a known issue.
- Avoid Perfetto, winter jacket, leg warmers and shoe covers unless the forecast
  changes toward rain, strong wind or much lower temperatures.
